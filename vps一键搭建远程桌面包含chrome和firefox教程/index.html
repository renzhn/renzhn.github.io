VPS一键搭建远程桌面（包含Chrome和Firefox）教程

December 18, 2020 · 1 min · root

前言#

推荐一个Docker镜像，用来一键搭建桌面环境（包含Chrome和Firefox），支持 VNC、RDP 协议，也可用浏览器（noVNC）直接打开。

这样可以在VPS或者群晖很方便地运行一个桌面环境，实现远程桌面连接！

VPS安装Docker#

使用官方一键安装脚本：

curl -Lv get.docker.com|bash

使用方法#

输入命令：

docker run -d -p 3389:3389 -p 5901:5901 -p 6901:6901 soff/tiny-remote-desktop

  • 浏览器（noVNC），直接访问：http://ip:6901，密码 vncpassword
  • VNC：ip:5901，密码 vncpassword
  • RDP：ip，用户名 user 密码 password

Credit: https://hub.docker.com/r/soff/tiny-remote-desktop
